# 객체지향
* 하나 이상의 유사한 객체들을 묶어서 하나의 공통된 특성을 포함하며, 
데이터를 추상화하는 단위로 사용되는 객체지향의 구성요소
-> 클래스

* 연관성 중 2개 이상의 객체가 서로 상호 관련있음을 의미하는 것
* is member of
-> 연관화

* 연관성 중 동일한 형의 특성을 같는 객체들을 모아 구성하는 것
* is instance of
-> 분류화

* 연관성 중 관련 있는 객체들을 묶어 하나의 상위 객체를 구성하는 것
* is part of
-> 집단화

* 연관성 중 공통적인 성질들로 추상화환 상위 객체들을 구성하는 것
-> 일반화

* 연관성 중 상위 객체를 구체화하여 하위 객체를 구성하는 것
-> 특수화
