# 배치 프로그램

* 배치 프로그램의 필수 요소 중, 대량의 데이터를 
가져오거나, 전달하거나, 계산하는 등 처리가 가능해야 함
-> 대용량 데이터

* 배치 프로그램의 필수 요소 중, 심각한 오류가 발생하는 상황을
제외하고는 사용자의 개입 없이 수행되어야 함
-> 자동화

* 오류가 발생하면 오류의 발생 위치, 시간 등을 추적할 수 
있어야 함
-> 안정성/신뢰성

* 다른 응용 프로그램의 수행을 방해하지 않아야 함
-> 성능

* 배치 프로그램이 일괄 처리 작업을 설정된 주기에 맞춰
원할히 수행하도록 지원하는 도구
-> 배치 스케쥴러
