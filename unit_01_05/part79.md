# 미들웨어 솔루션
* 운영체제와 응용 프로그램 사이, 또는 서버와 
클라 사이에서 다양한 서비스를 제공하는 소프트웨어
-> 미들웨어

* DB를 사용하여 시스템 구축 시 이를 ~라고 함
-> 2-Tier 아키텍처

* 응용 프로그램의 프로시저를 사용하여 원격 프로시저를
마치 로콜 프로시저처럼 호출
-> RPC

* 메시지 기반의 비동기형 메시지를 전달
* 온라인 업무보다는 이기종 분산 데이터 시스템의
데이터 동기를 위해 많이 사용
-> MOM

* 트랜잭션을 처리 및 감시
-> TP-Monitor

* 코바 표준 스펙을 구현
-> ORB

* 동적인 콘텐츠 처리
-> WAS
