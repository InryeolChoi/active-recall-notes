# 인터페이스 방법 명세화
## 인터페이스 통신 유형
* 시스템에서 거래를 요청하고 올 때까지 대기
-> 동기

* 시스템에서 거래를 요청하고 다른 작업을 수행하다
응답이 오면 처리하는 방식
-> 비동기

* 인터페이스 방법을 명세화 할 때 필요한 시스템
연계 기술 중 서버는 통신을 위해 포트를 할당하고 
클라이언트는 통신을 요청해 클라이언트와 연결하여
통신하는 네트워크 기술
-> 소켓

* 시스템 연계 기술 중 네트워크의 정보를 표준화된
서비스 형태로 만들어 공유하는 기술로, WSDL, UDDI
프로토콜을 이용하여 연계하는 기술
-> web service
